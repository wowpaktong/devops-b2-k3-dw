```
git clone // digunakan untuk mengcloning project dari server ke local
```
```
git init // digunakan untuk membuat git repository pada file lokal
```
```
git status // digunakan untuk mengetahui status dari repository lokal
```
```
git add // digunakan untuk menambahkan file baru pada repository yang akan dipilih
```
```
git commit // digunakan untuk menyimpan perubahan yang dilakukan pada local repository
```
```
git push // digunakan untuk mengirimkan perubahan file setelah di commit ke remote repository
```
